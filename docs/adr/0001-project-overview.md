# ADR-0001: プロジェクト概要と基本方針

## ステータス

承認済み

## コンテキスト

Emacsライクなテキストエディタを新規に開発する。
Emacsの基本概念（キーバインド、バッファ、ウィンドウ、モード）を踏襲しつつ、
Emacsのシングルスレッド設計など既知の課題を改善する。

## 決定

- プロジェクト名: **Allei** (アッレイ)
  - 由来: 古事記の編纂者・稗田阿礼(ひえだのあれ) + イタリア語で「できた」
- 実装言語: Java (JDK 21)
- TUIファーストで開発し、将来的にGUI対応可能な設計とする
- ビルドツール: Gradle (Kotlin DSL)
- 拡張言語: KTS (Kotlin Script)

## 帰結

- Java エコシステム上で動作し、拡張もJVM言語で書ける
- TUI/GUIの切り替えを可能にするため、レンダリング層を抽象化する必要がある
