# ADR-0006: レンダリング層の抽象化

## ステータス

承認済み

## コンテキスト

TUIファーストで開発するが、将来的にGUIにも対応したい。
入出力とレンダリングをエディタ本体から切り離す必要がある。

## 決定

- `allei-render-api` モジュールにレンダリングの抽象層（interface）を定義する
- TUI実装は `allei-tui` モジュールに配置し、JLine3ベースで実装する
- 将来のGUI実装は別モジュールとして追加する

## 帰結

- エディタ本体はレンダリングの詳細を知らずに動作する
- TUI/GUIの切り替えがモジュールの差し替えで可能になる

## 備考

未実装。TUIライブラリの選定として、JLine3のTerminal層を借りて自前の抽象レンダリング層を構築する方針。Lanternaのような高レベルフレームワークに依存しすぎるとGUI化時に剥がしにくくなるため。
