# ADR-0005: IO層の抽象化

## ステータス

承認済み

## コンテキスト

ファイルの読み書きは副作用を伴うため、テスタビリティと再現性のためにインターフェース経由で外部化する必要がある。

## 決定

- `BufferReader` / `BufferWriter` インターフェースを定義し、ファイルシステムへの直接依存を避ける
- `BufferIO` がこれらのインターフェース経由でファイル読み書きを行う
- テスト時にはインメモリ実装を差し替えて使用する
- 改行コードの検出・正規化・復元は `LineEnding` 列挙型が担当する
  - 読み込み時: 元の改行コードを検出し、LFに正規化してTextModelに格納
  - 保存時: 検出した改行コードに復元して書き出す

## 帰結

- 副作用が外部化され、テストが容易になる
- 改行コードの変換がIO層に集約される
